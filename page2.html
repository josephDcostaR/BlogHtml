História

O HTML surgiu com o objetivo de compartilhar documentos científicos entre pesquisadores. Criado por Tim Berners-Lee, ele permitiu a criação de links entre páginas, formando a base da World Wide Web.

Tags Estruturais

As tags estruturais são usadas para organizar e dar significado à estrutura de uma página HTML. Elas ajudam na organização do conteúdo, na semântica e são importantes para a acessibilidade e para mecanismos de busca (SEO).

Aqui estão as principais tags:

  • <body>: Contém todo o conteúdo visível da página.
  • <div>: Contêiner genérico, usado para agrupar elementos. Sem significado semântico por si só.
  • <header>: Representa o cabeçalho de uma seção ou da página. Normalmente contém logo, menu, título.
  • <main>: Define o conteúdo principal da página (único e principal).
  • <section>: Agrupa conteúdos com um mesmo tema ou propósito.
  • <article>: Conteúdo independente e reutilizável, como postagens ou notícias.
  • <aside>: Conteúdo secundário, como barras laterais ou sugestões.
  • <footer>: Rodapé de uma seção ou da página (informações de contato, copyright etc.).

Por que usar essas tags?

  • Semântica: Permitem que navegadores, leitores de tela e mecanismos de busca entendam melhor o conteúdo.
  • Organização: Facilitam a manutenção e a leitura do código.
  • SEO: Ajudam a posicionar melhor o site em mecanismos de busca.

Exemplo prático:

<body>
  <header>
    <h1>Meu Site</h1>
    <nav>
      <a href="#home">Início</a>
      <a href="#sobre">Sobre</a>
    </nav>
  </header>

  <main>
    <section>
      <h2>Bem-vindo</h2>
      <p>Conteúdo principal aqui.</p>
    </section>

    <aside>
      <p>Links úteis ou propaganda aqui.</p>
    </aside>
  </main>

  <footer>
    <p>© 2025 Joseph da Costa Ribeiro</p>
  </footer>
</body>

Tags Tabela

As tabelas HTML são usadas para organizar dados em linhas e colunas, sendo muito úteis para representar informações estruturadas. Abaixo estão as principais tags:

  • <table>: define o início de uma tabela.
  • <tr>: cria uma linha da tabela (table row).
  • <td>: cria uma célula de dado (table data).
  • <th>: cria uma célula de cabeçalho (table header), com texto em negrito e centralizado por padrão.
  • <thead>, <tbody>, <tfoot>: dividem a tabela semanticamente em cabeça, corpo e rodapé.
  • border: atributo usado (em HTML antigo) para exibir bordas. Hoje o ideal é usar CSS.

Exemplo simples:

<table>
  <tr>
    <th>Nome</th>
    <th>Idade</th>
  </tr>
  <tr>
    <td>Maria</td>
    <td>25</td>
  </tr>
</table>

Tags de Mídia

Essas tags permitem incorporar mídias como imagens, áudio e vídeo nas páginas HTML.

Imagem

Usando imagem local:

Imagem da internet (troca automaticamente ao recarregar a página):

<img src="midias/image.jpg" alt="gato" width="200" height="200">
<img src="https://picsum.photos/800/400" alt="Imagem Aleatória">
<img src="https://loremflickr.com/800/400/coffee" alt="Imagem de café">

Áudio

Áudio local:

Áudio da internet (exemplo do FreeSound):

<audio controls src="midias/musica.mp3"></audio>
<audio controls>
  <source src="https://www.soundhelix.com/examples/mp3/SoundHelix-Song-1.mp3" type="audio/mpeg">
</audio>

Vídeo

Vídeo local:

Vídeo da internet (exemplo com vídeo MP4 direto):

<video width="400" controls src="midias/video.mp4"></video>
<video width="400" controls>
  <source src="https://www.w3schools.com/html/mov_bbb.mp4" type="video/mp4">
</video>

Tags de Citação e Referência

Essas tags são usadas para marcar trechos de texto citados ou referenciados de outras fontes.

  • <blockquote>: para citações em bloco, geralmente longas.
  • <q>: para citações curtas, dentro de um parágrafo.
  • <cite>: para indicar a fonte de uma obra (livros, sites, músicas, etc.).
  • <abbr>: para abreviações com significado completo.

Exemplos:

<blockquote>:

A web é um ambiente em constante evolução, e o HTML é uma de suas fundações.

<q>: Conhecimento é poder.

<cite>: A Origem das Espécies por Charles Darwin.

<abbr>: Você conhece o significado de HTML?

<blockquote cite="https://developer.mozilla.org/pt-BR/">
  A web é um ambiente em constante evolução.
</blockquote>

<q>Conhecimento é poder.</q>

<cite>A Origem das Espécies</cite>

SEO

SEO (Search Engine Optimization) é o conjunto de técnicas que ajudam páginas a serem melhor indexadas por mecanismos de busca como o Google.

  • Uso correto de títulos (<h1>, <h2>...)
  • Metatags: <meta name="description">
  • Estrutura semântica
